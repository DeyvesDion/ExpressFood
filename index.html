ExpressFood

Base de données d'une start-up pour une application de restauration en ligne.
La realisation de ce projet m'a permis d'acquérir les compétences suivantes:

  • Réaliser des schémas UML cohérents et en accord avec les besoins du client
  • Concevoir l’architecture technique d’une application à l’aide de diagrammes UML
  • Implémenter le schéma de données dans la base
  • Réaliser un schéma de conception de la base de données de l’application
